# 代码审查命令

执行全面的代码质量检查，涵盖后端Java代码和前端Vue代码。

## 执行步骤

1. **分析项目结构**
   - 检查后端Spring Boot项目结构
   - 检查前端Vue 3项目结构
   - 识别核心模块和组件

2. **后端代码审查 (Java)**
   - 检查Controller层的RESTful API设计
   - 检查Service层的业务逻辑
   - 检查Mapper/DAO层的数据访问
   - 检查实体类和DTO设计
   - 检查异常处理机制
   - 检查Shiro权限配置
   - 识别潜在的NPE、资源泄漏等问题

3. **前端代码审查 (Vue 3)**
   - 检查组件结构和Composition API使用
   - 检查API请求封装
   - 检查状态管理(Pinia)
   - 检查路由配置
   - 检查表单验证和用户体验

4. **安全检查**
   - SQL注入风险
   - XSS防护
   - 认证授权漏洞
   - 敏感信息泄露

5. **性能检查**
   - 数据库查询优化
   - 缓存使用
   - 前端性能

## 输出格式

提供结构化的审查报告，包括：
- 问题等级（严重/警告/建议）
- 问题位置（文件路径）
- 问题描述
- 改进建议
- 代码示例
